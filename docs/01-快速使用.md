# 快速使用

这个工具用于把异环的 DLSS 渲染比例写到 33% 以下。它是本机工具，不是网站服务。

## 准备

- Windows 10/11。
- NVIDIA RTX 显卡和可用的 DLSS。
- 已安装异环。
- 源码运行需要 Python 3；如果使用打包版 exe，则不需要安装 Python。

## 启动

打包版：

```text
双击 NTEDLSSPanel.exe
```

如果 HUD 开关提示没有权限：

```text
双击 run_exe_as_admin.bat
```

源码版：

```text
双击 run.bat
```

如果要使用 HUD 开关但提示没有权限：

```text
双击 run_as_admin.bat
```

浏览器打开：

```text
http://127.0.0.1:22532
```

## 安装

1. 点“浏览”。
2. 选择异环安装根目录，例如 `<游戏安装目录>`。
3. 点“检测路径”。
4. 选择比例，默认 `0.30`。
5. 保持“四个 DLSS 档位使用同一个比例”开启。
6. 点“按当前档位设置安装 / 更新”。
7. 完全退出游戏和启动器。
8. 重新进游戏并打开 DLSS。

## 验证

推荐开启 NVIDIA DLSS HUD：

1. 面板里点“开启 HUD”。
2. 重启游戏。
3. 看屏幕角落的 DLSS 输入/输出分辨率。

4K 输出下常见结果：

```text
0.30  -> 约 1152x648
0.315 -> 约 1210x680
0.333 -> 约 1280x720
```

如果 HUD 仍显示约 720p，先回到面板刷新日志，确认 `dlsstweaks.log` 是否存在。

成功日志应包含：

```text
WINMM.dll wrapper loaded
DLSS functions found & parameter hooks applied
```

## 恢复

面板中选择备份，然后点“恢复所选备份”。恢复会读取 `manifest.json`，按安装前状态还原或移除文件。
