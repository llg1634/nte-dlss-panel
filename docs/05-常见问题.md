# 常见问题

## 页面打不开

确认 `run.bat` 窗口没有报错，然后打开：

```text
http://127.0.0.1:22532
```

如果端口被占用，关闭旧的 Python 进程后重试。

## 点浏览没有弹窗

浏览按钮会调用 Windows 文件夹选择器。如果浏览器阻止或后端没有启动，可以手动复制游戏路径到输入框。

示例：

```text
E:\Neverness To Everness
```

## 检测不到游戏

工具会寻找：

```text
Client\WindowsNoEditor\HT\Binaries\Win64\HTGame.exe
```

可以直接选择 `HTGame.exe` 所在的 `Win64` 文件夹。

## 日志没有生成

常见原因：

- 游戏没有完全重启。
- `winmm.dll` 没有放到 `HTGame.exe` 同目录。
- 被安全软件隔离。
- 游戏更新后加载链变化。

先确认 Win64 目录有：

```text
winmm.dll
dlsstweaks.ini
```

然后完整退出游戏和启动器再进。

## HUD 显示还是 720p

先刷新日志。如果日志没有 `DLSS functions found & parameter hooks applied`，说明 DLSSTweaks 没有 hook 成功。

如果日志显示 hook 成功，但 HUD 仍是 720p，确认 `dlsstweaks.ini` 里 `[DLSSQualityLevels]` 是否启用，并且 `Performance` 等档位是否是目标比例。

## NVIDIA 全局比例要不要开

不需要。

本工具用局部 `dlsstweaks.ini` 修改。NVIDIA `GlobalForcedScale` 曾经测试过，但低于约 0.333 会被限制到 720p，因此不作为最终方案。

## HUD 开关会影响画质吗

不会。HUD 只显示调试信息，不改变 DLSS 渲染比例。用完可以在面板里关闭。

## 如何完全还原

1. 打开面板。
2. 选择游戏路径。
3. 选择安装前对应的备份。
4. 点“恢复所选备份”。
5. 如果开过 HUD，点“关闭 HUD”。
