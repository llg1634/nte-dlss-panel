## 功能验证

```powershell
python -m py_compile app.py
```

手动验证：

- `run.bat` 能启动。
- 首页能打开。
- `/api/state` 正常。
- `/api/shutdown` 能关闭本地后端服务。
- 游戏路径检测正常。
- 安装前会创建备份。
- 恢复会按 manifest 回滚。
- HUD 状态能读取。
