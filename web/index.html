04

安装步骤

面板会按这个顺序执行，状态区会显示实际结果。

1. 定位目录 找到 HTGame.exe 所在 Win64 文件夹。
2. 创建备份 备份到 Win64\_nte_dlss_backups\时间戳。
3. 清理旧入口 删除旧的 dxgi.dll / nvngx.dll / XInput1_4.dll。
4. 写入新配置 写入 winmm.dll 和 dlsstweaks.ini。
修改文件会在检测成功后显示。正常情况下只会新增或替换 winmm.dll 与 dlsstweaks.ini。
